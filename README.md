# Python Microservice Template

## Getting started

- [Install cookiecutter](https://cookiecutter.readthedocs.io/en/latest/installation.html#install-cookiecutter)

- Start a new project:
  `cookiecutter https://github.com/adolya/fastapi-cookiecutter-template.git//templates/python`

- Change dir to a newly create app

- Start application:
  `python .`
